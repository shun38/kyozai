PHPとは

PHPはHTMLのソースコードをサーバー側で生成し、ユーザーに渡すプログラムです。
phpタグを組み込み、主にhtmlのソースコードを 何かのデータを元に、サーバー側で作成する部分の仕事をしてくれます。
拡張子が.phpのファイルを作り、以下の様にPHPのソースコードを記述すると、ユーザ側ではhtmlファイルとして2つ目のソースコードの様に表示されます
・サーバー側のソースコード
  1. <html>
  2. <meta charset="utf-8">
  4. <body>
  5. <?php
  6. print('バナナ<br>');
  7. ?>
  8. </body>
  10. </html>
・クライアント側で認識されるのソースコード
  1. <html>
  2. <meta charset="utf-8">
  4. <body>
  5. バナナ<br>
  6. </body>
  8. </html>
phpファイルを作成したら、XAMPPやlinuxなどで動かします。

XAMPPでローカル環境を作る

XAMPPはローカル環境です。apacheを動作させたらブラウザを開いて、URL欄にlocalhost/ファイル名.phpと入力するとその指定したファイルを読み込むことができます。
XAMPPをインストールしたときに指定したパスにxamppというフォルダが作られ、htdocsというフォルダが中にあります。その中にphpファイルを置くことでphpファイルを読み取らせることが出来ます。
htdocsの中にもフォルダを作ることが可能です。その場合パスの指定は、
  1. localhost/folder/index.php
という様に指定します。
XAMPPでは別のPCからページを読み取ることはできません

Linuxでウェブサーバーを作る

Linuxではウェブサーバーソフト(今回使うのはapache)とPHPをインストールし、var/www/htmlの下にファイルを配置し、IPを指定してブラウザから読み取ります。 設定をすることで、別のPCから接続ができます。
Linux
PHPの中身はクライアント側からは見ることができません。
クライアントユーザー側には必ずhtmlソースが表示されます。
PHPではデータベースと接続したりクッキーやセッションを扱うことができます。
データベースと接続することによって、データの保存、抽出、それを基にページを生成する事ができます。
勿論自分でプログラムを考えてそれらを扱うことで、出来ることならできます。
これからPHPで具体的には何ができるかを知り、それを扱って便利なシステムを作ってみましょう！

演算処理
変数は以下のように同じ名前で複数指定すれば、沢山出現させられます。
例えば1.1という数が消費税の計算に使う数だった場合、変数名で英語で税という意味のtaxを指定することでそれが税金だという事をコードを見る人に伝えられます。
しかし、その英単語を知らない場合は、調べなければ理解できません。英語を勉強しておくとソースコードを読むときや変数名を決める時に便利です。
  1. <?php
  2. $length=5;#変数の宣言
  3. $outdata=$length*$length;#outdataという変数の中にlength変数*length変数の数を格納
  4. print($outdata."\n");#5*5の演算をして結果は25
  6. $outdata=$length*$length*$length;#再び変数に代入するコードを記述すれば何回でも代入できます。
  7. print($outdata."\n");#25*5の演算をして結果は125
  9. $outdata+=$length;#+=とすることで、左の変数に、そのまま右の変数の中身が加算されます。
  10. print($outdata."\n");#125+5の演算をして結果は130
  12. $outdata-=$length;#-=とすることで、左の変数に、そのまま右の変数の中身が減算されます。*=や/=や%=も同じです。
  13. print($outdata."\n");#130-5の演算をして結果は125
  15. $outdata++;#++とすると1が加算されます。--では1が減産されます。PHPでは**がありません。+=1を短くできます。
  16. print($outdata."\n");#125+1の演算をして結果は126
  18. ?>
ちなみに、変数名の指定にはルールがあります。PHPでは、数字で開始するとエラーが発生し、日本語の文字で指定したり、namaeやzeikinという様な 名前を指定するのは望ましくありません。
userNameという様に英単語を用いて、2語以上の場合は、大文字で区切ると読みやすく望ましい変数名です。

データベース接続
PFO (PHP Data Objects)とは、PHPからデータベースにアクセスする機能を提供する拡張モジュールです。
データベース固有のPDOドライバが使用され、接続するデータベースにかかわらず、同じ操作でデータにアクセスできます。
PDOクラスは、PHPとデータベースサーバーの間の接続を表します。
PDOクラスのインスタンスを作成することにより、データベースサーバーとの接続が確立されます。
以下のコードでは、PDO::__constructメソッドを使用してインスタンス(データ構造などの展開をし、実行できるようにしたもの)を生成します。
コンストラクタなので、直接呼び出すコードは書かずに、new演算子を用います。
  1. <?php
  2. $pdo = new PDO($dsn, $username, $password, $driver_options);
  3. ?>

  40. ?>
(1)でmysqliオブジェクトのインスタンスを作成することでデータベースに接続します。
変数$mysqliには接続に成功するとデータベースへの接続状態を含めたmysqliオブジェクトが入り、接続に失敗したときはfalseが入ります。

(2)ではデータを取得するSQLをプリペアドステートメントを使って登録します。
SQLのWHERE句にある「?」はプレースホルダーと呼び、続く(3)でbind_paramメソッドを使って検索したいidカラムの値をセットします。

(4)でここまで用意したSQLクエリを実行して、(5)で取得したデータの値を各変数で受け取る設定をします。
データを受け取る変数はテーブルのカラムに応じて用意する必要があります。

(6)でfetchメソッドを実行すると、(5)で設定した変数に取得したデータが先頭から順にカラムごとにセットされていきます。　

(7)のcloseメソッドでデータベースとの接続を解除して終了です。

(8)の箇所では(1)~(7)でエラーが起こったときにcatch文で受け取ってエラーメッセージを出力します。
mysqli関数の使用例
  1. <?php
  3. $id = 10;
  4. $host = 'localhost';

  7. $db = 'test';

  11.   // (1) データベースに接続

  23.   // (5) 取得したデータをセットする変数を設定

  26.   // (6) 結果を取得
  27.   if( mysqli_stmt_fetch($stmt) ) {
  28.   var_dump( $id, $first_name, $last_name );
  29.   }
  31.   // (7) データベースの接続解除

全体的な流れはオブジェクトを使ったときと同様です。
大きな違いはオブジェクトのメソッドではなく関数のみ使用するため、接続情報の入ったmysqliオブジェクトやプリペアドステートメントのmysqli_stmtオブジェクトは変数を経由して関数に渡して実行する必要があります。
こちらの例では(1)で作成したmysqliオブジェクトを変数$mysqliに入れて、mysqli_stmtオブジェクトは変数$stmtに入れてそれぞれ関数に渡して実行しています。
